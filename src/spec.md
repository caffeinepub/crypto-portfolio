# Specification

## Summary
**Goal:** Add a transaction history list to display all buy and sell transactions.

**Planned changes:**
- Create a backend function that returns all user transactions with symbol, company name, type, quantity, price, timestamp, and total value
- Add a new transaction history section below the archived holdings panel
- Display transactions in a table with columns for all transaction details
- Show Buy/Sell badges to distinguish transaction types
- Sort transactions by date with newest first

**User-visible outcome:** Users can view a complete history of all their buy and sell transactions in a dedicated section below their archived holdings.
